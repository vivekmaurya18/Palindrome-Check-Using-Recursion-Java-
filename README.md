# 🔁 Palindrome Check Using Recursion (Java)

## 📌 Problem Statement

Check whether a given string is a **palindrome** using **recursion**.

A palindrome is a string that reads the same forward and backward.
Examples: `MADAM`, `ABBBA`, `LEVEL`

---

## 💡 Approach

We use recursion to compare characters from both ends of the string:

* Compare the **first** and **last** characters
* If they are equal → move inward
* Repeat until:

  * All characters match ✅ → Palindrome
  * Any mismatch ❌ → Not a palindrome
